# 디자인 시스템

기준 원본은 [[../99_보관/2026-06-14_루트_정리/UI_초안/DESIGN|DESIGN.md]]의 Part A다. 이 문서는 구현자가 빠르게 확인할 수 있게 핵심만 압축한다.

## 컨셉

터미널/CLI 감성의 판타지 RPG.

- 검은 배경
- 인광 그린 포인트
- 모노스페이스 폰트
- 스캔라인/CRT 느낌
- 실시간 로그와 커서 깜빡임

하지 말 것:

- 둥근 모서리 카드
- 파스텔/그라데이션 배경
- 이모지 남발
- 일반 산세리프 UI

## 전역 규칙

```css
body {
  background:var(--bg);
  color:var(--txt);
  font-family:var(--mono);
  font-size:14px;
  line-height:1.5;
  overflow:hidden;
  word-break:keep-all;
}
```

한국어 UI에서는 `word-break:keep-all`을 필수로 둔다.

## 레이아웃

- 좌측 레일: `236px`
- 상단바: `60px`
- 콘텐츠는 앱 셸 내부에서 스크롤한다.
- 패널/버튼/행은 1px 보더와 각진 모서리를 기본으로 한다.
- 기본 패널 패딩은 `14px`.
- 카드 간격은 보통 `14px`.

## 공통 컴포넌트

- 패널: `.panel`
- 버튼: `.btn`
- 진행 바: `.bar`
- 큰 숫자 지표: `.metric`
- 라벨/값 행: `.kv`
- 탭: `.tabs`, `.tab`
- 리스트 행: `.row`
- 태그/뱃지: `.tag`
- 장비/정수 슬롯: `.slot`
- 모달: `.scrim`, `.modal`
- 토스트: `#toasts`, `.toast`

## 전투 로그 색상

원칙:

- 액센트 그린: 신호, 치명타, 정수 발동 강조, 시스템, 보상 수치
- 빨강: 피해
- 보라: 정수
- 등급 라벨: 등급색
- 일반 공격: 눈에 덜 띄는 회색

로그 범례는 피드 위에 항상 표시한다.

```txt
일반 / 치명타 / 정수 / 드랍 / 처치
```

## CRT 효과

화면 전체에 고정 오버레이를 둔다.

- 스캔라인
- 비네팅
- 미세 플리커
- 커서 깜빡임

모션 규칙:

- 항상 보이는 콘텐츠의 진입 애니메이션은 `opacity`를 건드리지 않는다.
- 콘텐츠 애니메이션은 `transform` 중심으로 처리한다.
- 모달/토스트처럼 사용자 액션으로 뜨는 요소는 opacity 페이드 가능.
- `prefers-reduced-motion: reduce`에서는 애니메이션을 끈다.
