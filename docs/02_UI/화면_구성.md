# 화면 구성

기준 원본은 [[../99_보관/2026-06-14_루트_정리/UI_초안/DESIGN|DESIGN.md]] Part B와 [[../99_보관/2026-06-14_루트_정리/UI_초안/rpg-tower-design-brief|rpg-tower-design-brief.md]]다. Part B는 예시이며, 디자인 시스템은 [[디자인_시스템]]을 따른다.

## 페이지 목록

| 페이지 | 역할 |
|---|---|
| 인증 | 로그인/회원가입 |
| 대시보드 | 현재 상태 허브 |
| 사냥 | 사냥터 선택, 자동사냥 현황 |
| 탑 | 층 목록, 수동 도전, 전투 결과 |
| 캐릭터 | 스탯 확인/분배 |
| 장비 | 장비 장착/비교 |
| 정수 | 정수 슬롯, 목록, 합성 |
| 랭킹 | 최고 도달 층수 순위 |

인증은 필요하지만 최대한 간결하게 구현한다.

## 대시보드

표시 정보:

- 캐릭터 이름, 레벨, 경험치
- 현재 사냥터와 사냥 상태
- 누적 골드
- 최고 도달 탑 층수
- 최근 드랍 요약
- 빠른 이동 버튼

## 사냥

사냥터 선택:

- 진입 가능한 사냥터 목록
- 잠긴 사냥터 표시
- 권장 레벨
- 등장 몬스터
- 추천 사냥터 태그
- 자동사냥 시작/정지

사냥 현황:

- 실시간 전투 로그
- 드랍 목록
- 현재 체력
- 정수 쿨타임
- 휴식 기준 체력 설정

## 탑

층 목록:

- 1~100층
- 5층 단위 구간 표시
- 클리어 여부
- 입장 비용
- 잠금 여부
- 보스층 강조
- 랭킹 현황

층 입장/전투:

- 입장 비용 확인
- 전투 로그
- 장착 정수 슬롯과 쿨타임
- 클리어 결과
- 드랍 정수

보스 최초 클리어 보상은 확정 정수다.
업적 보상은 현재 없다.

## 캐릭터

- 1차 스탯 7종
- 잔여 스탯 포인트
- 스탯 분배 UI
- 2차 스탯 계산값
- 스탯 재분배

## 장비

슬롯:

- 무기
- 방어구
- 장신구

기능:

- 현재 장착 장비
- 보유 장비 목록
- 옵션 상세
- 장비 비교
- 등급 색상 표시

장비 강화는 추후 업데이트 예정이다.

## 정수

슬롯:

- 정수 슬롯 3개
- 액티브/패시브 타입 표시
- 액티브 정수 쿨타임 표시

목록:

- 보유 정수 전체
- 타입 필터
- 등급 필터
- 계열 필터
- 등급별 효과 비교

합성:

```txt
같은 정수 3개 -> 상위 등급 정수 1개
```

## 랭킹

- 최고 도달 층수 기준
- 내 순위 고정 표시
- 순위, 닉네임, 최고 층수, 달성 일시

## 단축키

| 키 | 이동 |
|---|---|
| `1` | 대시보드 |
| `2` | 사냥 |
| `3` | 탑 |
| `4` | 캐릭터 |
| `5` | 장비 |
| `6` | 정수 |
| `7` | 랭킹 |
| `Tab` | 다음 탭/섹션 |
| `Esc` | 뒤로 / 모달 닫기 |
