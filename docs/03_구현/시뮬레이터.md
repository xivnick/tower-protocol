# 시뮬레이터

기준 원본은 [[../99_보관/2026-06-14_루트_정리/초안/rpg-tower-sim|rpg-tower-sim.md]]다.

## 목적

몬스터, 정수, 장신구 수치를 조정하기 위한 CLI/웹 시뮬레이터.

현재 목적은 실제 게임 코드가 아니라 밸런스 감각 확인이다.

같은 `--seed`를 쓰면 같은 전투 로그가 나오므로 수치 변경 전후 비교에 사용할 수 있다.

## 실행

### 신규 100층 밸런스 시뮬레이터

`rpg-tower-balance-sim.mjs`는 [[04_게임_데이터/01_정수_목록|정수 목록]] 문서를 기준으로 밸런스 검토용 전투를 돌린다.
기존 구현 데이터와 분리되어 있으므로, 새 100층 설계의 수치 감각을 먼저 확인하는 용도다.

효과 파싱 확인:

```bash
node rpg-tower-balance-sim.mjs inspect --floor 8
```

단일 층 전투:

```bash
node rpg-tower-balance-sim.mjs duel --floor 10 --build str --level 10 --seed 7 --log
```

플레이어 정수 직접 지정:

```bash
node rpg-tower-balance-sim.mjs duel --floor 80 --build mage --level 180 --essences 8,31,62 --seed 7
```

탑 구간 표:

```bash
node rpg-tower-balance-sim.mjs tower --build tank --from 1 --to 20 --levelOffset 0 --seed 7
```

현재 지원하는 주요 효과:

- 직접 물리/마법/방어 기반 피해
- 다음 일반 공격 강화
- 다음 정수 피해 강화
- 추가타 자세
- 독, 화상, 출혈, 둔화, 동결, 속박, 무력화
- 방어막
- 회복, 흡혈, 재생
- 공격력, 방어, 회피, 치명타, 공격속도, 정수 피해 버프
- 표식/독/화상/둔화/감전/방어막/낮은 HP 조건부 피해
- 행동 지연, 쿨타임 감소

주의:

- 이 시뮬레이터는 밸런스 검토용 근사 모델이다.
- 정수 사용은 일반공격 게이지를 소모하지 않는 무료 스킬이다.
- 한 틱에 한 캐릭터는 행동 하나만 수행한다.
- 정수가 준비되어 있으면 해당 틱에는 정수 하나를 사용하고, 일반공격 게이지가 차 있어도 일반공격은 다음 틱 이후로 밀린다.
- 플레이어 기본 로드아웃은 빌드별 대표 정수 3개를 사용한다.
- 문서 문장 파싱 기반이므로, 효과 문장을 크게 바꾸면 `inspect`로 파싱 결과를 먼저 확인한다.

### 기존 구현 시뮬레이터

공용 데이터는 `rpg-tower-data.js`에 둔다.
CLI와 웹 뷰어는 같은 몬스터/정수/장신구 데이터를 읽는다.

목록 출력:

```bash
node rpg-tower-sim.mjs list
```

전투 로그 포함 1:1 전투:

```bash
node rpg-tower-sim.mjs duel --build flurry --level 10 --enemy forestWolf --rank elite --enemyLevel 5 --accessory serratedRing --log --seed 7
```

매트릭스:

```bash
node rpg-tower-sim.mjs matrix --enemy forestBoar --seed 3
```

탑 진행:

```bash
node rpg-tower-sim.mjs tower --build str --level 100 --from 1 --to 100 --seed 7
```

맞춤 전략:

```bash
node rpg-tower-sim.mjs tower --strategy counter --level 100 --from 60 --to 80 --seed 7
```

## 주요 옵션

| 옵션 | 설명 |
|---|---|
| `--build` | `str`, `flurry`, `tank`, `mage` |
| `--strategy` | `counter`를 주면 상대에 맞춰 빌드/장신구/정수를 자동 선택 |
| `--level` | 플레이어 레벨 |
| `--accessory` | 장신구 키 |
| `--essences` | `wolfFlurry:2,bladeEdge:2,lizardScale:2` 형식 |
| `--enemy` | 몬스터 키 |
| `--rank` | `normal`, `elite`, `boss` |
| `--enemyLevel` | 몬스터 레벨 |
| `--log` | 전투 로그 출력 |
| `--seed` | 난수 고정값 |

## 기존 구현에 반영된 규칙

- 10틱 = 1초
- 공격속도는 `sqrt(공격속도 / 100)`으로 초당 일반공격 게이지를 계산한다.
- 정수 사용은 일반공격 게이지를 소모하지 않는 무료 스킬이다.
- 한 캐릭터는 한 틱에 행동 하나만 수행한다.
- 같은 캐릭터에게 정수와 일반공격이 동시에 준비되어 있으면 정수 슬롯 순서가 일반공격보다 우선한다.
- 일반 몬스터는 전투에서 2단계 정수를 사용하고, 드랍은 1단계 정수다.
- 엘리트와 보스는 전투에서 4단계 정수를 사용하고, 드랍은 1~3단계 정수다.
- 플레이어는 1~5단계 정수를 장착해 비교할 수 있다.
- 물리/마법 피해는 명중과 회피 판정을 받는다.
- 고정 피해, 독, 출혈은 방어와 치명타 영향을 받지 않는다.
- 장신구는 전투 효과 1개를 기준으로 처리한다.

## 맞춤 전략

| 전략 | 용도 |
|---|---|
| 기본 | 특별한 위협이 낮은 상대 |
| 대마법 | 지력/지혜가 높은 마법형 상대 |
| 흡혈 돌파 | 기사, 야수, 보스, 강한 물리 압박 상대 |
| 명중 연격 | 회피/손재주가 높은 상대 |
| 파쇄 마법 | 방어/체력이 높은 상대 |
| 생존 | 장기전 생존 확인용 |
