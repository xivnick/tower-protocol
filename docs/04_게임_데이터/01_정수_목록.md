# 정수 목록

이 문서는 정수의 등급별 효과와 몬스터별 특징을 정리한다.

## 작성 기준

- 각 정수는 1등급부터 5등급까지 모두 확인한다.
- 효과 문장은 게임 안에서 실제로 적용되는 규칙을 기준으로 쓴다.
- 의미가 애매한 효과는 확정하기 전에 이 문서에서 먼저 정리한다.

## 훈련장 · 인공 정수

인공 정수는 훈련장에 낮은 확률로 등장하는 정수 착용 허수아비가 드랍한다.
정수를 착용한 허수아비를 처치하면 해당 인공 정수 1등급을 확정 획득한다.

일반 정수는 등급이 오를수록 종합 전투 기여가 크게 상승하고 4~5등급에 빌드 특수효과가 붙는다.
인공 정수는 훈련용 기본 정수이므로 등급별 체감 효율을 1등급 기준 약 2~5배로 완만하게 잡고, 특수효과도 기본 전투 축 안에서만 붙인다.
