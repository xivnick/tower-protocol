# TOWER:// 문서 홈

웹 기반 자동사냥 RPG `TOWER://`의 정리된 기획 문서 모음.

## 문서 관리 원칙

- 최신 기준은 이 홈에서 연결한 핵심 문서를 따른다.
- 논의로 기준이 바뀌면 관련 최신 문서를 즉시 갱신한다.
- 같은 내용을 중복 문서로 유지하지 않는다.
- 충돌하거나 폐기된 내용은 삭제하거나 `99_보관`으로 내린다.
