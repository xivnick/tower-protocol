# RPG Tower — 디자인 시안 브리프

## 서비스 개요

웹 기반 자동사냥 RPG. 핵심 루프는 **사냥 → 장비/정수 강화 → 탑 수동 도전**이다.
자동사냥은 오프라인에도 진행되며, 탑은 플레이어가 직접 수동으로 도전한다.
랭킹은 최고 도달 층수 기준이다.

---

## 로고 및 아이덴티티

**로고:** `TOWER://`

- 터미널/CLI 감성. 프롬프트나 URL 스킴처럼 읽히는 느낌.
- 타입페이스는 모노스페이스 계열 권장.
- 색상 방향은 디자이너에게 위임 (다크 배경 + 포인트 컬러 조합 자유롭게).
- 커서 깜빡임, 스캔라인, 글리치 같은 CRT 레퍼런스를 레이어로 얹을 수 있음.

---

## 플랫폼

- **데스크탑 우선** — 키보드 단축키로 페이지 이동 포함
- **모바일 반응형** — 데스크탑 레이아웃을 기반으로 모바일 대응

---

## 페이지 목록

### 1. 인증 (로그인 / 회원가입)

- 로그인, 회원가입 폼
- 서비스 로고와 분위기를 처음 전달하는 첫 화면
- 별도 페이지 2개 또는 탭 전환

---

### 2. 대시보드 (메인)

서비스 진입 후 기본 화면. 현재 상태를 한눈에 파악하는 허브.

표시 정보:
- 캐릭터 이름, 레벨, 현재 경험치
- 현재 사냥 중인 사냥터 이름 + 상태 (사냥 중 / 사망 패널티 중 / 정지)
- 누적 골드, 최고 도달 탑 층수
- 최근 드랍 아이템 요약 (몇 개, 어떤 종류)
- 빠른 이동 버튼 (사냥 / 탑 / 장비 / 정수)

---

### 3. 사냥

두 단계로 구성.

**3-1. 사냥터 선택**
- 진입 가능한 사냥터 목록 (잠긴 사냥터는 흐리게)
- 각 사냥터: 이름, 권장 레벨, 등장 몬스터 종류
- 현재 추천 사냥터에 태그 표시
- 선택 후 자동사냥 시작/정지 토글

**3-2. 사냥 현황**
- 현재 전투 로그 피드 (실시간 스크롤)
- 드랍 아이템 목록
- 현재 체력, 정수 쿨타임 상태
- 회복 설정 (휴식 기준 체력 % 조정)

---

### 4. 탑

**4-1. 탑 층 목록**
- 1~100층을 구간(5층 단위)으로 묶어 표시
- 각 층: 클리어 여부, 입장 비용 (골드), 잠금 여부
- 보스 층(5의 배수)은 별도 강조
- 랭킹 현황 (현재 내 순위)

**4-2. 층 입장 / 전투**
- 입장 비용 확인 후 도전 시작
- 전투 로그 실시간 표시
- 사용 중인 정수 슬롯과 쿨타임 상태
- 클리어 결과: 드랍 정수
- 보스 최초 클리어 보상: 확정 정수
- 업적 보상은 현재 없음

---

### 5. 캐릭터

- 1차 스탯 7종 표시 (근력 / 민첩 / 손재주 / 체력 / 인내 / 지력 / 지혜)
- 잔여 스탯 포인트 및 분배 UI
- 2차 스탯 계산값 (물리 공격력, 방어력, 회피 등) 펼쳐보기
- 스탯 재분배 버튼 (재화 소모)

---

### 6. 장비

슬롯: 무기 / 방어구 / 장신구 (각 1개)

- 현재 장착 장비와 보유 장비 목록
- 장비 클릭 시 옵션 상세 표시
- 강화 UI (강화 버튼, 현재 강화 단계, 소모 재료)
- 장비 비교 (장착 중 vs 선택한 장비)
- 장비 등급 색상 구분

---

### 7. 정수

**7-1. 슬롯 배치**
- 정수 슬롯 3개 (드래그 또는 클릭으로 배치)
- 슬롯별 타입 표시 (액티브 / 패시브)
- 액티브 정수는 쿨타임 수치 표시

**7-2. 정수 목록**
- 보유 정수 전체 목록
- 필터: 타입(액티브/패시브), 등급, 계열(몬스터 이름)
- 정수 상세: 등급별 효과 비교 테이블

**7-3. 합성**
- 같은 정수 3개 → 상위 등급 합성
- 합성 가능 목록 하이라이트

---

### 8. 랭킹

- 최고 도달 층수 기준 순위표
- 내 순위 고정 표시 (리스트 중간이라도)
- 각 항목: 순위, 닉네임, 최고 층수, 달성 일시

---

## 키보드 단축키 (데스크탑)

| 키 | 이동 |
|----|------|
| `1` | 대시보드 |
| `2` | 사냥 |
| `3` | 탑 |
| `4` | 캐릭터 |
| `5` | 장비 |
| `6` | 정수 |
| `7` | 랭킹 |
| `Tab` | 다음 탭/섹션 |
| `Esc` | 뒤로 / 모달 닫기 |

---

## 디자인 요청 사항

- 로고 `TOWER://` 의 타입, 색상, 애니메이션 방향 제안
- 전체 컬러 시스템 제안 (다크 배경 기반 + 포인트 컬러)
- 페이지 간 공통 레이아웃 (네비게이션 위치, 헤더 구조)
- 전투 로그 피드의 타이포그래피 및 색상 구분 방식 (일반 공격 / 크리티컬 / 정수 발동 / 드랍 등)
- 정수 등급별 색상 체계 (1~5등급)
- 모바일 반응형 레이아웃 방향
