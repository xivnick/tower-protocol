## 핵심 구조

- 장르: 웹 기반 자동사냥 RPG
- 핵심 루프: 사냥 -> 장비/정수/골드/경험치 획득 -> 캐릭터 성장 -> 탑 수동 도전
- 자동사냥은 게임을 꺼둔 동안에도 진행된다.
- 탑은 수동 도전 콘텐츠이며, 1층부터 100층까지 권장 성장선에 맞춰 진행한다.
- 만렙: 100
- 탑: 100층 목표
- 사냥터: 20개, 각 사냥터는 5레벨 구간 담당
- 탑 난이도: 레벨과 1:1 대응하지 않으며, 60층은 100레벨 권장선, 100층은 200레벨급 엔드콘텐츠
- 사냥터 몬스터는 해당 탑 5층 구간의 1~4층 몬스터의 normal 버전이며, 탑에서는 같은 몬스터가 elite 버전으로 등장한다.
- 탑은 `1~4층 엘리트 + 5층 보스` 구간을 반복한다.
- 탑 5층짜리 구간 1개는 사냥터 1개와 연결된다.
- 탑 5의 배수 층은 해당 사냥터 구간 boss가 등장한다.
- 레벨업마다 스탯 포인트 +5

## 스탯

### 1차 스탯

| 스탯 | 역할 |
|------|------|
| 근력 | 물리 공격 |
| 민첩 | 공격속도, 회피 |
| 손재주 | 명중, 치명타 확률 |
| 체력 | 최대 체력 |
| 인내 | 물리 방어, 재생 |
| 지력 | 마법 공격 |
| 지혜 | 마법 방어, 쿨타임 감소 |

### 1차 -> 2차 변환

| 1차 스탯 1 | 2차 스탯 증가 |
|------------|----------------|
| 근력 1 | 물리 공격 +1 |
| 민첩 1 | 공격속도 +1, 회피 +1 |
| 손재주 1 | 명중 +1, 치명타 확률 +1% |
| 체력 1 | 최대 체력 +10 |
| 인내 1 | 물리 방어 +1, 재생 +1 |
| 지력 1 | 마법 공격 +1 |
| 지혜 1 | 마법 방어 +1, 쿨타임 수치 +1 |

### 기본값

```txt
기본 1차 스탯: 각 10
기본 공격속도: 100
기본 명중: 100
기본 회피: 0
기본 치명타 확률: 0%
기본 치명타 피해: 150%

최대 체력 = 100 + 레벨 × 20 + 체력 × 10
최종 방어 = 물리 방어 + 마법 방어
```

### 3차 계산식

공격속도:

```txt
초당 공격 횟수 = sqrt(공격속도 / 100)
틱당 일반공격 게이지 = 초당 공격 횟수 / 10
```

회피:

```txt
빗나감 확률 = 회피 / (회피 + 명중)
```

치명타:

```txt
치명타 확률 = min(치명타 확률, 100%)
기본 치명타 피해 = 150%
초과 치명타 확률은 기본적으로 버려짐
초과 치명타 확률 활용은 장신구/정수 효과로 처리
```

방어:

```txt
피해 배율 = 100 / (100 + 최종 방어)
피해 감소율 = 최종 방어 / (100 + 최종 방어)
```

쿨타임 감소:

```txt
쿨타임 감소율 = 쿨타임 수치 / (쿨타임 수치 + 300)
최종 쿨타임 = 기본 쿨타임 × (1 - 쿨타임 감소율)
```

재생:

```txt
초당 체력 회복 = 최대 체력 × (재생 / 10000)
```

피해:

```txt
물리 피해 = 기본 위력 × (1 + 물리 공격 / 100)
마법 피해 = 기본 위력 × (1 + 마법 공격 / 100)
```

## 전투

- 10틱 = 1초
- 일반공격 게이지가 1 이상이면 일반공격을 실행한다.
- 물리 피해와 마법 피해 모두 명중/회피 판정을 받는다.
- 고정 피해, 반사 피해, 지속 피해는 치명타가 발생하지 않는다.
- 매 틱마다 양쪽의 일반공격 게이지와 정수 쿨타임은 모두 갱신된다.
- 한 캐릭터는 한 틱에 행동을 최대 1개만 실행할 수 있다.
- 액티브 정수 사용은 일반공격 게이지를 소모하지 않는 무료 행동이다.
- 단, 정수도 그 틱의 행동 1개로 취급하므로 같은 캐릭터가 같은 틱에 일반공격까지 같이 실행하지는 않는다.
- 같은 캐릭터에게 여러 행동이 준비되어 있으면 정수 슬롯 순서가 일반공격보다 우선한다.

동일 시점 우선순위:

1. 플레이어 정수
2. 플레이어 일반공격
3. 몬스터 정수
4. 몬스터 일반공격

예:

```txt
전투 시작 시 플레이어 정수 3개와 몬스터 정수가 모두 준비되어 있어도
0.0초: 플레이어 1번 정수 -> 몬스터 정수
0.1초: 플레이어 2번 정수 -> 몬스터 일반공격 가능 시 실행
0.2초: 플레이어 3번 정수 -> 몬스터 행동

즉, 준비된 행동을 한 틱에 모두 쏟아붓지 않는다.
```

일반공격 속도 보정:

```txt
최종 틱당 일반공격 게이지 = 기본 틱당 일반공격 게이지 × 일반공격 속도 배율
```

- 둔화/가속 같은 전투 중 속도 보정은 공격속도 수치를 직접 바꾸지 않고 일반공격 속도 배율로 적용한다.
- 일반공격 속도 보정은 곱연산으로 중첩한다.
- 예: 20% 둔화 2개 = 0.8 × 0.8 = 0.64배

## 사냥 회복 루틴

공통 회복 액션 시간:

```txt
RECOVERY_ACTION_SECONDS = 1
```

휴식:

```txt
자동사냥에서 전투 종료 후 자동 사냥 설정에 따라 실행
RECOVERY_ACTION_SECONDS만큼 시간 소요
휴식 후 체력은 항상 100%까지 회복
액티브 정수 쿨타임도 RECOVERY_ACTION_SECONDS만큼 감소
```

포션:

```txt
휴식 실행 시점에 포션을 보유 중이면 자동 사용
포션 사용은 휴식을 한 것으로 취급
RECOVERY_ACTION_SECONDS만큼 시간 소요
체력은 100%까지 회복
액티브 정수 쿨타임도 RECOVERY_ACTION_SECONDS만큼 감소
```

- 포션은 휴식이 필요해지는 순간 자동으로 소모되어 사냥 흐름을 안정화한다.
- 포션을 충분히 보유하면 위험한 사냥터에서도 휴식 기준에 걸린 뒤 바로 회복할 수 있어 사냥 효율을 높일 수 있다.

사망 패널티:

```txt
기본 10초
사망 패널티 중 체력과 액티브 정수 쿨타임 회복
```

치유감소는 전투 중 회복에만 적용한다.

적용:
- 재생
- 흡혈
- 회복 정수

미적용:
- 휴식
- 포션
- 사망 패널티 중 회복

## 정수

- 정수는 몬스터가 실제 전투에서 사용하는 능력을 플레이어가 얻어 장착하는 시스템이다.
- 정수 이름에는 몬스터 이름을 붙인다.
- 액티브 정수와 패시브 정수가 있으며, 총 3개 슬롯을 공유한다.
- 액티브 정수는 쿨타임마다 자동 발동한다.
- 패시브 정수는 장착 중 항상 적용된다.
- 액티브 정수 쿨타임은 주로 3~7초 범위에 둔다.
- 보스 정수는 긴 쿨타임의 강한 한 방보다 전투 스타일을 바꾸는 고유 효과를 우선한다.

등급:

| 등급 | 이름 | 역할 |
|------|------|------|
| 1등급 | 희미한 | 능력을 보여주는 맛보기 |
| 2등급 | 온전한 | 초반 실사용 |
| 3등급 | 응축된 | 일반적인 사용 기준 |
| 4등급 | 정제된 | 고성능 |
| 5등급 | 각성한 | 추가 효과가 붙는 완성형 |

합성:

```txt
같은 정수 3개 -> 다음 등급 정수 1개
다른 몬스터의 정수는 합성 재료로 섞지 않음
정수 강화가 상위 몬스터 정수로 변하지 않음
```

쿨타임 시작:

- 자동사냥은 연속 사냥 타임라인으로 처리하며, 전투가 끝나도 남은 쿨타임이 다음 전투로 이어진다.
- 자동사냥을 처음 시작할 때는 장착한 액티브 정수가 준비된 상태로 시작한다.
- 탑 도전과 보스전은 전투 시작 시 장착한 액티브 정수가 준비된 상태로 시작한다.

## 상태이상과 특수 효과

### 상태이상

상태이상은 기본적으로 스택 없이 지속시간만 가진다.
수치는 상태이상 자체가 아니라 부여한 스킬/아이템이 정한다.

| 상태이상 | 효과 |
|----------|------|
| 취약 | 받는 피해 증가 |
| 약화 | 주는 피해 감소 |
| 실명 | 명중 감소 |
| 둔화 | 행동속도 감소 |
| 파쇄 | 최종 방어 감소 |
| 치유감소 | 전투 중 받는 회복량 감소 |

공통 규칙:

```txt
명중/회피 판정 성공 시 부여
같은 상태이상을 다시 부여하면 더 강한 수치가 우선
수치가 같거나 낮으면 지속시간만 갱신
```

### 독

독은 공통 상태이상이나 스택형 시스템이 아니라, 정수/아이템 효과에 붙는 지속 피해로 처리한다.

```txt
스택 없음
피해량은 부여한 정수/아이템이 정함
방어 영향 없음
치명타 없음
```

예:

```txt
독버섯 슬라임의 독포자 정수
- 5초간 초당 고정 피해 8

독침 반지
- 치명타 시 5초간 초당 고정 피해 5
```

### 출혈

출혈은 스택형 특수 효과다.

```txt
스택당 매초 대상 현재 체력 비례 피해
방어 영향 없음
치명타 없음
체력이 낮아질수록 피해량도 낮아짐
마무리 피해보다는 체력이 많은 적을 빠르게 약하게 만드는 역할
```

예:

```txt
출혈 1스택 = 매초 현재 체력의 0.1% 피해
최대 20스택
20스택 = 매초 현재 체력의 2% 피해
```

### 스택형 정수 효과

스택형은 일반 상태이상이 아니라 특정 정수/아이템의 고유 메커니즘으로 처리한다.

예:

```txt
냉기: 5스택 도달 시 1초간 빙결
잔불: 3스택 도달 시 폭발
표식: 일정 스택 도달 시 다음 공격 강화
의식: 일정 스택 도달 시 회복/정수 강화
```

## 장비

장비 철학:

```txt
스탯은 기반을 만들고
장비는 그 기반을 증폭하거나 전투 방식을 바꾼다
일반 장비 옵션은 퍼센트 중심
빌드를 바꾸는 특수 옵션은 고정값/특수 규칙 가능
```

장비 등급:

| 등급 | 이름 | 레벨 구간 |
|-----:|------|-----------|
| 1 | 낡은 | 0~20 |
| 2 | 보통의 | 20~40 |
| 3 | 정교한 | 40~60 |
| 4 | 뛰어난 | 60~80 |
| 5 | 완벽한 | 80~100 |

- 같은 이름의 장비라도 등급에 따라 수치 범위가 달라진다.
- 같은 등급 안에서도 실제 획득 아이템은 범위 안에서 조금씩 다른 수치를 가질 수 있다.
- 퍼센트 옵션은 보통 1% 단위로, 고정값 옵션은 1 단위로 굴린다.

### 무기

무기 밸런스는 평타 DPS만으로 보지 않는다.
각 무기 타입에 맞는 정수 세팅을 포함해 전투 시간과 생존 여부로 평가한다.

```txt
물리 무기: 근력형 정수 포함
마법 무기: 마법 공격력을 사용하는 정수 포함
원거리 무기: 연격/다단 타격 정수 포함
```

물리 무기:

```txt
물리 공격 +%
근력/물리 피해 빌드용
```

마법 무기:

```txt
마법 공격 +%
지력/마법 정수 빌드용
```

원거리 무기:

```txt
물리 공격 낮음
명중 감소
타격마다 고정 피해 +N
연격/공속/손재주 빌드용
```

- 원거리 무기는 비율형 고정 피해가 아니라 `+X 고정 피해`만 사용한다.
- 명중 감소 수치와 고정 피해 수치는 아이템 등급/개체에 따라 달라질 수 있다.
- 강한 원거리 무기일수록 명중 감소가 크고, 타격마다 고정 피해 수치도 높다.
- 높은 등급 원거리 무기를 쓰려면 손재주/명중 세팅이 필요하다.
- 원거리 무기의 고정 피해는 일반공격뿐 아니라 정수의 각 타격에도 적용한다.
- 예: `명중 -65%, 타격마다 고정 피해 +120` 무기는 명중 세팅이 충분한 후반 연격 빌드용이다.

### 방어구

방어구는 체력을 직접 늘리지 않는다.
체력은 캐릭터 스탯 또는 장신구/특수 옵션에서 얻는다.

중갑:

```txt
물리 방어 +%
반사 피해 또는 받는 피해 감소 계열
```

- 중갑의 물리 방어는 항상 퍼센트다.
- 방벽형 중갑은 물리 방어 +%와 모든 피해 감소를 가진다.
- 모든 피해 감소는 방어 계산 이후 적용되며, 고정 피해/독/출혈/반사 피해도 일부 줄이는 역할이다.
- 가시형 중갑은 물리 방어 +%와 반사 피해를 가진다.
- 일반 방어구는 최대 체력을 직접 늘리지 않는다.

경갑:

```txt
물리 방어 +고정값 또는 +%
회피 +고정값 또는 +%
```

- 경갑은 물리 방어와 회피를 둘 다 가진다.
- 각 옵션은 고정값일 수도 있고 퍼센트일 수도 있다.
- 초반에는 고정값 옵션의 체감이 좋고, 후반에는 퍼센트 옵션의 성장성이 좋다.

로브:

```txt
마법 방어 +고정값 또는 +%
쿨타임 수치 +고정값 또는 +%
마법 공격은 올리지 않음
```

- 로브는 마법 방어와 쿨타임 수치를 둘 다 가진다.
- 각 옵션은 고정값일 수도 있고 퍼센트일 수도 있다.
- 로브는 마법 공격을 직접 올리지 않고, 정수 회전율과 마법 방어 경로를 지원한다.

물리 방어와 마법 방어는 최종 방어에 합산되어 모든 피해에 대응한다.
즉, 어떤 공격을 맞는지가 아니라 어떤 경로로 방어 수치를 쌓는지가 중요하다.

### 장신구

장신구는 스탯 1개 또는 특수 효과 1개 정도로 명확하게 설계한다.
장신구도 장비 등급을 사용하며, 같은 장신구라도 등급과 개체에 따라 수치가 달라진다.

기본 퍼센트 장신구 범위:

| 등급 | 이름 | 수치 범위 |
|-----:|------|-----------|
| 1 | 낡은 | 4~8% |
| 2 | 보통의 | 8~12% |
| 3 | 정교한 | 12~16% |
| 4 | 뛰어난 | 16~20% |
| 5 | 완벽한 | 20~24% |

- 장신구 이름은 효과 종류를 정한다.
- 등급은 등장 구간과 수치 범위를 정한다.
- 실제 획득 장신구는 해당 범위 안에서 랜덤 수치를 가진다.
- 특수 장신구는 효과 자체는 고정하고, 수치 부분만 등급 범위를 가진다.

예:

```txt
탐욕의 동전
- 골드 획득량 +%

정수 보관함
- 정수 드랍률 +%

거인의 심장
- 최대 체력 +%

얼음 심장
- 마법 공격력의 일부를 최대 체력으로 추가

매의 눈
- 명중 +%

바람 깃털
- 회피 +%

붉은 송곳니
- 물리 공격 +%

푸른 수정구
- 마법 공격 +%

현자의 모래시계
- 쿨타임 수치 +%

날카로운 표식
- 치명타 확률 +%

정밀한 조준경
- 초과 치명타 확률의 50%를 치명타 피해로 전환

독침 반지
- 치명타 시 5초간 초당 고정 피해를 주는 독 부여

톱니 반지
- 치명타 시 출혈 1스택 부여
```

## 탑 드랍

- 탑 입장 비용은 실제 게임 규칙이다.
- 층별 비용 공식은 아직 정하지 않았다.
- 일반 몬스터는 전투에서 2등급 정수를 사용하고, 처치 시 1등급 정수를 드랍한다.
- 엘리트 몬스터와 보스 몬스터는 전투에서 4등급 정수를 사용한다.
- 탑 엘리트 몬스터의 정수 드랍률은 10%다.
- 보스 최초 클리어 시 보스 정수 1개를 확정 지급한다.
- 탑에서 드랍되는 정수는 1~3등급 중 랜덤이다.
- 업적 보상은 현재 없다.

탑 정수 등급 확률:

| 등급 | 확률 |
|------|------|
| 1등급 | 50% |
| 2등급 | 30% |
| 3등급 | 20% |
