# RPG Tower Essence Balance

## 기준

- 10틱 = 1초
- 일반 사냥: 권장 레벨 + 평균 장비로 안정적으로 처치 가능한 수준
- 탑 일반층: 권장 레벨에서 클리어 가능한 수준
- 탑 보스층: 권장 레벨에서 클리어 가능한 수준
- 액티브 정수 쿨타임 기준: 3~7초
- 보스 정수는 긴 쿨타임의 강한 한 방보다 전투 스타일을 바꾸는 독특한 효과를 우선한다.
- 보스 정수도 액티브, 패시브, 조건부 발동 중 하나로 설계할 수 있다.
- 일반 사냥터 몬스터는 1등급 정수를 드랍한다.
- 탑 엘리트 몬스터는 같은 정수를 10% 확률로 드랍한다.
- 탑 보스는 고유 정수를 드랍한다.
- 보스 최초 클리어 시 보스 정수 1개를 확정 지급한다.
- 탑에서 드랍되는 정수는 1~3등급 중 랜덤이다.
- 탑 정수 등급 확률은 1등급 50%, 2등급 30%, 3등급 20%다.

정수 등급:

| 등급 | 이름 | 역할 |
|------|------|------|
| 1등급 | 희미한 | 능력을 보여주는 맛보기 |
| 2등급 | 온전한 | 초반 실사용 |
| 3등급 | 응축된 | 일반적인 사용 기준 |
| 4등급 | 정제된 | 고성능 |
| 5등급 | 각성한 | 추가 효과가 붙는 완성형 |

정수 쿨타임 시작 규칙:

- 일반 자동사냥은 연속 사냥 타임라인으로 처리하며, 전투가 끝나도 남은 쿨타임이 다음 전투로 이어진다.
- 자동사냥을 처음 시작할 때는 장착한 액티브 정수가 준비된 상태로 시작한다.
- 탑 도전과 보스전은 전투 시작 시 장착한 액티브 정수가 준비된 상태로 시작한다.

휴식 규칙:

- 자동사냥에서 전투가 끝나면 자동 사냥 설정에 따라 휴식 여부를 판단한다.
- 공통 회복 액션 시간은 `RECOVERY_ACTION_SECONDS = 1`로 둔다.
- 휴식은 `RECOVERY_ACTION_SECONDS`만큼 시간을 소모한다.
- 휴식을 취하면 체력이 100%까지 회복된 상태로 다음 전투를 시작한다.
- 휴식 중 액티브 정수 쿨타임도 `RECOVERY_ACTION_SECONDS`만큼 감소한다.
- 휴식 실행 시점에 포션을 보유 중이면 포션을 자동 사용한다.
- 포션 사용은 휴식을 한 것으로 취급하며, 같은 시간 소모와 정수 쿨타임 감소가 적용된다.
- 포션은 위험한 사냥터에서 휴식 타이밍의 회복을 보장해 사냥 효율을 높이는 소모품이다.
- 사망 패널티는 기본 10초이며, 이 시간 동안 체력과 액티브 정수 쿨타임도 회복된다.

수치 표기:

- 물리 피해/마법 피해의 `%`는 해당 피해 계열의 공격력 또는 스킬 위력 계수다.
- 고정 피해는 방어력과 치명타 계산 이후 타격마다 더해지는 방어 무시 피해다.
- 지속 피해는 별도 표기가 없으면 1초마다 적용한다.
