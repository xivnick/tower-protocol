## 유지할 만한 초안 주제

- 보석 시스템
- 장비 강화 세부 규칙
- 업적 보상
- 스탯 재분배 비용
- 탑 입장 비용
- 1~30층 상세 구간별 학습 의도

이 항목들은 바로 폐기하지 않고, 필요할 때 확정 문서로 승격한다.
