# 정수

정수는 몬스터가 사용하는 능력을 플레이어가 획득해 빌드에 편입하는 성장 시스템이다.

## 기본 개념

- 몬스터를 처치하면 낮은 확률로 정수 획득
- 정수에는 해당 몬스터의 특징이 녹아있는 스킬이 있음
- 몬스터도 전투 중 자신에게 대응되는 정수 스킬을 사용
- 플레이어는 정수를 장착해 전투 방식을 바꿀 수 있음

## 장착

- 정수 슬롯: 3개
- 액티브 정수와 패시브 정수가 같은 슬롯을 공유
- 액티브 정수는 쿨타임이 돌아오면 자동 사용
- 패시브 정수는 장착 중 항상 적용
- 액티브 정수도 고등급에서 장착 중 적용되는 패시브 효과나 특수 효과를 함께 가질 수 있음

## 등급

정수는 1등급부터 5등급까지 존재한다.

| 등급 | 역할 |
|---:|---|
| 1등급 | 첫 발동으로 능력의 기본 형태를 체험하는 단계 |
| 2등급 | 전투 중 반복 사용감이 생기는 단계 |
| 3등급 | 빌드 후보로 들어오는 실사용 단계 |
| 4등급 | 엘리트와 보스 몬스터가 사용하는 빌드 시너지 단계 |
| 5등급 | 해당 빌드에 극단적으로 투자했을 때 정수 하나가 1인분에 가까워지는 최종 단계 |

## 밸런스 기준

- 5등급 정수는 해당 정수에 맞는 스탯과 장비를 극단적으로 맞췄을 때, 같은 레벨 캐릭터 1명분에 가까운 전투 기여를 목표로 한다.
- 5등급 정수 3개를 갖춘 종결 세팅은 본체를 포함해 4인분보다 조금 낮은 전투력을 목표로 한다.
- 등급 상승은 총 전투 기여가 약 2배씩 강해지는 느낌을 목표로 한다.
- 이때 2배는 단일 수치가 아니라 피해량, 회복량, 방어막, 지속시간, 쿨타임, 조건 완화, 부가 효과, 세팅 시너지를 합친 체감 기준이다.
- 전투 시작 시 장착한 액티브 정수는 한 번씩 사용된다.
- 1등급은 쿨타임이 길어도 전투 시작 첫 발동으로 정수의 성격을 보여줄 수 있다.
- 2등급부터는 쿨타임 감소나 효과량 증가를 통해 전투 중 반복 사용 체감을 만들 수 있다.
- 쿨타임 감소는 등급 성장 방식 중 하나다.
- 등급 상승 방식은 정수 성격에 따라 효과량 증가, 지속시간 증가, 조건 완화, 부가 효과 추가, 쿨타임 감소 중에서 고른다.

## 효과 서술

- 높은 등급 정수는 낮은 등급의 핵심 효과를 유지한 채 강화된다.
- 4~5등급에서 부가 효과가 생기더라도 기존 등급의 핵심 기능은 이어진다.
- 각 등급 설명은 해당 등급의 실제 효과를 모두 적는다.
- 하위 등급을 함께 읽어야만 이해되는 생략 표현은 쓰지 않는다.

## 합성

- 같은 정수 3개를 합치면 다음 등급 정수 1개 획득
- 서로 다른 몬스터의 정수는 합성 재료로 섞지 않음
- 정수 합성은 같은 정수의 등급만 올림

```txt
같은 1등급 정수 3개 -> 같은 정수 2등급 1개
같은 2등급 정수 3개 -> 같은 정수 3등급 1개
같은 3등급 정수 3개 -> 같은 정수 4등급 1개
같은 4등급 정수 3개 -> 같은 정수 5등급 1개
```

## 몬스터 사용 규칙

- 일반 사냥 몬스터는 전투에서 2등급 정수를 사용
- 탑 엘리트 몬스터는 전투에서 4등급 정수를 사용
- 탑 보스 몬스터는 전투에서 4등급 정수를 사용

## 드랍

- 사냥터 일반 몬스터는 1등급 정수 드랍
- 탑 엘리트 몬스터와 보스 몬스터는 1~3등급 정수 드랍
- 보스 최초 클리어 보상은 확정 보스 정수
