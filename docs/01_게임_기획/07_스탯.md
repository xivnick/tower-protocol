# 스탯

스탯은 캐릭터의 기본 성장 방향을 정하는 시스템이다.
레벨업으로 얻은 스탯 포인트를 1차 스탯에 분배하고, 1차 스탯에서 전투에 사용하는 2차 스탯을 계산한다.

## 성장 규칙

- 레벨업마다 스탯 포인트를 5 얻는다.
- 스탯 포인트는 1차 스탯에 분배한다.
- 기본 1차 스탯은 모두 10이다.
- 스탯 재분배는 가능하며, 재화를 소모한다.

## 1차 스탯

| 스탯 | 영문 | 역할 |
|---|---|---|
| 근력 | STR | 물리 공격 |
| 민첩 | AGI | 공격속도, 회피 |
| 손재주 | DEX | 명중, 치명타 |
| 체력 | VIT | 최대 체력 |
| 인내 | END | 물리 방어, 재생 |
| 지력 | INT | 마법 공격 |
| 지혜 | WIS | 마법 방어, 쿨타임 감소 |

## 1차 스탯 계산

1차 스탯은 기본값, 분배값, 장비/정수/버프 보너스를 합쳐 계산한다.

```txt
최종 1차 스탯 = (기본 1차 스탯 + 분배 스탯 + 1차 스탯 고정 보너스) × (1 + 1차 스탯 퍼센트 보너스)
```

- 같은 계층의 퍼센트 보너스는 합산한다.
- 최종 1차 스탯은 2차 스탯 계산의 기준값이 된다.

예:

```txt
기본 STR 10
분배 STR 90
장비 STR +20
정수 STR +10
버프 STR +20%

최종 STR = (10 + 90 + 20 + 10) × 1.20 = 156
```

## 1차 스탯에서 2차 스탯 변환

| 1차 스탯 1 | 2차 스탯 증가 |
|---|---|
| 근력 1 | 물리 공격 +1 |
| 민첩 1 | 공격속도 +1, 회피 +1 |
| 손재주 1 | 명중 +1, 치명타 확률 +1% |
| 체력 1 | 최대 체력 +10 |
| 인내 1 | 물리 방어 +1, 재생 +1 |
| 지력 1 | 마법 공격 +1 |
| 지혜 1 | 마법 방어 +1, 쿨타임 수치 +1 |

기본값:

```txt
기본 공격속도 = 100
기본 명중 = 100
기본 회피 = 0
기본 치명타 확률 = 0%
기본 치명타 피해 = 150%
```

최대 체력:

```txt
최대 체력 = 100 + 레벨 × 20 + 체력 × 10
```

최종 방어:

```txt
최종 방어 = 물리 방어 + 마법 방어
```

## 2차 스탯 계산

2차 스탯은 1차 스탯 변환값에 2차 스탯 보너스를 적용해 계산한다.

```txt
기본 2차 스탯 = 최종 1차 스탯 기반 공식
최종 2차 스탯 = (기본 2차 스탯 + 2차 스탯 고정 보너스) × (1 + 2차 스탯 퍼센트 보너스)
```

예:

```txt
최종 STR 156
STR 1당 물리 공격 +1
무기 물리 공격 +50
버프 물리 공격 +30%

최종 물리 공격 = (156 + 50) × 1.30 = 267.8
```

## 3차 계산식

공격속도:

```txt
초당 공격 횟수 = sqrt(공격속도 / 100)
틱당 일반공격 게이지 = 초당 공격 횟수 / 10
```

회피:

```txt
빗나감 확률 = 회피 / (회피 + 명중)
```

치명타:

```txt
치명타 확률 = min(치명타 확률, 100%)
기본 치명타 피해 = 150%
```

- 초과 치명타 확률은 기본적으로 버려진다.
- 초과 치명타 확률 활용은 장신구나 정수 효과로 별도 처리한다.

방어:

```txt
피해 배율 = 100 / (100 + 최종 방어)
피해 감소율 = 최종 방어 / (100 + 최종 방어)
```

쿨타임 감소:

```txt
쿨타임 감소율 = 쿨타임 수치 / (쿨타임 수치 + 300)
최종 쿨타임 = 기본 쿨타임 × (1 - 쿨타임 감소율)
```

재생:

```txt
초당 체력 회복 = 최대 체력 × (재생 / 10000)
```

피해:

```txt
일반 공격의 물리 피해 = 최종 물리 공격력
정수가 주는 물리 피해 = 기본 위력 × (1 + 물리 공격 / 100)
마법 피해 = 기본 위력 × (1 + 마법 공격 / 100)
```

## 적용 순서

스탯 계산은 1차 스탯, 2차 스탯, 전투 결과 순서로 처리한다.

1. 기본 1차 스탯
2. 레벨업으로 분배한 1차 스탯
3. 장비, 정수, 버프의 1차 스탯 고정 증가
4. 장비, 정수, 버프의 1차 스탯 퍼센트 증가
5. 최종 1차 스탯 확정
6. 최종 1차 스탯을 기반으로 2차 스탯 계산
7. 장비, 정수, 버프의 2차 스탯 고정 증가
8. 장비, 정수, 버프의 2차 스탯 퍼센트 증가
9. 최종 2차 스탯 확정
10. 전투 공식 적용
11. 최종 피해, 회복, 쿨타임 등 결과값 보정
12. 상한/하한 적용

## 빌드 방향

두 개 스탯을 집중 육성하는 것이 기본 방향이다.
비율은 1:1 또는 2:1 정도를 기준으로 한다.
한쪽만 극단적으로 올리면 공격이나 생존 중 하나가 부족해질 수 있다.

| 빌드 | 생존 방식 | 처치 방식 |
|---|---|---|
| 근력 중심 | 체력, 방어 일부 | 강한 물리 피해 |
| 민첩 + 손재주 | 회피, 빠른 행동 | 연격, 치명타, 타격당 고정 피해 |
| 지력 + 지혜 | 마법 방어, 쿨타임 운용 | 마법 피해, 지속 피해, 스택형 정수 |
| 체력 + 인내 | 높은 체력, 방어, 재생 | 반사 피해, 방어력 기반 피해 |
| 인내 + 지혜 | 방어, 재생, 약화 | 디버프 누적, 재생/반사 연계 피해 |

지혜는 직접 피해를 크게 올리는 스탯이 아니라 정수 쿨타임 감소를 통해 전투 효과를 빠르게 누적하는 운용형 스탯이다.
스택형 정수는 지혜 빌드의 핵심 대상이다.
