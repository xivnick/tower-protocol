# 밸런스

## 기준

- 10틱 = 1초
- 일반 사냥 몬스터는 권장 레벨 + 평균 장비로 안정적으로 처치 가능해야 한다.
- 탑 일반층은 권장 레벨에서 클리어 가능해야 한다.
- 탑 보스층은 권장 레벨에서 클리어 가능해야 한다.
- 전투 시간은 결과로 관찰하되, 고정 목표로 삼지 않는다.
- 권장 레벨보다 낮으면 실패하고, 권장선에 맞추면 클리어되는 것을 기준으로 한다.

## 정수 밸런스

- 액티브 정수 쿨타임 기준: 3~7초.
- 보스 정수는 긴 쿨타임의 강한 한 방보다 전투 스타일을 바꾸는 독특한 효과를 우선한다.
- 정수는 1~5등급 데이터를 가진다.
- 시뮬레이터에서는 피해, 버프, 회복, 지속 피해, 상태이상, 명중/회피, 치명타, 반사, 쿨타임 환급 효과 위주로 구현한다.

정수 등급:

| 등급 | 이름 | 역할 |
|---|---|---|
| 1등급 | 희미한 | 능력을 보여주는 맛보기 |
| 2등급 | 온전한 | 초반 실사용 |
| 3등급 | 응축된 | 일반적인 사용 기준 |
| 4등급 | 정제된 | 고성능 |
| 5등급 | 각성한 | 추가 효과가 붙는 완성형 |

|---|---|

| 엘리트 몬스터 | 4등급 |

| 탑 엘리트 | 같은 몬스터 정수 1~3등급, 10% |

## 경험치와 골드

레벨업 필요 경험치:

```txt
다음 레벨 필요 경험치 = ceil((다음 레벨^1.5 × 50) / 100) × 100
```

- `다음 레벨`은 레벨업 후 도달할 레벨이다.
- 최종값은 100 단위로 올림 처리한다.
- LV.100이 최고 레벨이며, LV.100에서는 레벨업 경험치를 더 이상 요구하지 않는다.
- 이 공식은 선형 증가보다 빠르고 2차 증가보다 완만한 성장 곡선을 만든다.
- 저레벨 사냥터 반복 사냥 방지는 레벨업 요구 경험치가 아니라 몬스터 경험치 보정에서 처리한다.

예시:

| 현재 레벨 | 다음 레벨 | 필요 경험치 |
|---:|---:|---:|
| 1 | 2 | 200 |
| 2 | 3 | 300 |
| 4 | 5 | 600 |
| 9 | 10 | 1,600 |
| 14 | 15 | 3,000 |
| 19 | 20 | 4,500 |
| 24 | 25 | 6,300 |
| 49 | 50 | 17,700 |
| 74 | 75 | 32,500 |
| 99 | 100 | 50,000 |

몬스터 경험치와 골드 초안:

권장 공식:

```txt
일반 몬스터 경험치 = round(10 + 몬스터 레벨 × 4.5)
일반 몬스터 골드 = round(5 + 몬스터 레벨 × 2.2)

강한 개체 경험치 = 일반 경험치 × 1.25
강한 개체 골드 = 일반 골드 × 1.25
```

몬스터 경험치와 골드 공식은 [[../99_보관/2026-06-14_루트_정리/초안/rpg-tower-monster-item-design|rpg-tower-monster-item-design.md]]의 초안이며 아직 최신 규칙으로 확정되지는 않았다.

## 몬스터 조정 초안

현재 시뮬레이터 관찰:

| 대상 | 현재 감각 | 조정 방향 |
|---|---|---|
| 일반 몬스터 | 3~5초대가 자주 나와 빠름 | 체력 또는 방어를 소폭 증가 |
| 엘리트 몬스터 | 7~15초대가 많아 짧음 | 체력 증가, 정수 위협 유지 |
| 보스 | 피해가 과하면 즉사감이 생김 | 권장 레벨 기준 생존 가능하게 조정 |

추천 계수 초안:

```txt
일반 몬스터 HP: 현재 기준 × 1.25
엘리트 몬스터 HP: 현재 기준 × 1.45
보스 몬스터 HP: 현재 기준 × 1.20
보스 공격력: 현재 기준 × 0.90
```

## 시뮬레이션 해석

| 대상 | 클리어 조건 |
|---|---|
| 일반 사냥 몬스터 | 권장 레벨 + 평균 장비로 안정적으로 처치 가능 |
| 탑 1~60층 | 100레벨 기본 빌드가 60층 전후까지 진행 가능 |
| 탑 61~100층 | 100레벨 이후 장비/정수 성장으로 미는 엔드 콘텐츠 |

현재 기준 체크:

```bash
node rpg-tower-sim.mjs tower --build str --level 100 --from 55 --to 70 --seed 7
```

기본 근력 빌드, 장신구 없음 기준으로 60층 보스는 클리어하고 75층부터 첫 실패가 나온다.

맞춤 전략 체크:

```bash
node rpg-tower-sim.mjs tower --strategy counter --level 100 --from 64 --to 75 --seed 7
```

맞춤 전략은 상대별로 다른 세팅을 고르며, 기본 근력으로 막히는 일부 층을 통과한다.
