# 콘텐츠 구조

## 전체 구조

- 사냥터: 20개
- 탑: 100층
- 각 사냥터는 5레벨 구간을 담당한다.
- 같은 몬스터는 사냥터에서는 normal, 탑에서는 elite 버전으로 등장한다.
- 탑은 `1~4층 엘리트 + 5층 보스` 구조를 반복한다.
- 탑 5층짜리 구간 1개는 사냥터 1개와 연결된다.
- 따라서 100층 탑은 20개 사냥터와 대응한다.

### 자동사냥

- 사냥터에서는 자동사냥을 진행
- 전투에 소모된 시간만큼 실제 사냥 시간이 흐름
- 자동사냥 중 체력과 액티브 정수 쿨타임은 다음 전투로 이어짐
- 전투 종료 후 휴식 설정에 따라 휴식 여부 판단
- 사냥터에서는 장비, 골드, 경험치, 1등급 정수 드랍

### 휴식과 포션

공통 회복 액션 시간:

```txt
RECOVERY_ACTION_SECONDS = 1
```

휴식:

- 전투 종료 후 휴식 조건을 만족하면 실행
- `RECOVERY_ACTION_SECONDS`만큼 시간 소모
- 휴식 후 체력은 100%까지 회복
- 휴식 중 액티브 정수 쿨타임도 `RECOVERY_ACTION_SECONDS`만큼 감소

포션:

- 휴식 실행 시점에 포션을 보유 중이면 자동 사용
- 포션 사용은 휴식을 한 것으로 취급
- 포션 사용 시에도 `RECOVERY_ACTION_SECONDS`만큼 시간 소모
- 포션 사용 시에도 액티브 정수 쿨타임이 `RECOVERY_ACTION_SECONDS`만큼 감소
- 포션 사용 후 체력은 100%까지 회복

사망 패널티:

- 자동사냥 중 체력이 0이 되면 사망
- 사망 시 기본 10초간 사냥 정지
- 사망 패널티 중 체력과 액티브 정수 쿨타임 회복
- 사망 패널티 후 자동사냥 재개

## 탑

탑은 1~100층이다.

탑 등급 규칙:

```txt
각 5층 구간의 1~4층: 해당 사냥터 일반 몬스터의 elite 버전
각 5층 구간의 5층: 해당 사냥터 보스
```

예:

```txt
1~4층: 1번 사냥터 엘리트
5층: 1번 사냥터 보스
6~9층: 2번 사냥터 엘리트
10층: 2번 사냥터 보스
```

탑 레벨 스케일:

```txt
1~10층: 상대 레벨 = 층
11~30층: 상대 레벨 = 10 + ceil((층 - 10) × 1.5)
31~50층: 상대 레벨 = 40 + ceil((층 - 30) × 2)
51~60층: 상대 레벨 = 80 + ceil((층 - 50) × 2)
61~100층: 상대 레벨 = 100 + ceil((층 - 60) × 2.5)
```

중요 지점:

| 층 | 상대 레벨 | 의미 |
|---:|---:|---|
| 5 | 5 | 1번 사냥터 보스 |
| 10 | 10 | 2번 사냥터 보스 |
| 30 | 40 | 초기 완성도 테스트 |
| 50 | 80 | 얼어붙은 능선 보스 |
| 60 | 100 | 만렙 권장선 |
| 100 | 200 | 엔드 콘텐츠 |

## 탑 입장 비용

탑 입장 비용은 실제 게임 규칙이다.

아직 층별 비용 공식은 정하지 않았다.

## 클리어 보상

- 보스 최초 클리어 보상은 확정 보스 정수다.
- 업적 보상은 현재 없다.

## 빌드 방향

| 빌드 | 생존 방식 | 처치 방식 |
|---|---|---|
| 근력 중심 | 체력, 방어 일부 | 강타, 괴력, 강한 물리 피해 |
| 민첩 + 손재주 | 회피, 빠른 행동 | 연격, 치명타, 타격당 고정 피해 |
| 지력 + 지혜 | 마법 방어, 쿨타임 운용 | 마법 피해, 지속 피해, 스택형 정수 |
| 체력 + 인내 | 높은 체력, 방어, 재생 | 반사 피해, 방어력 기반 피해 |
| 인내 + 지혜 | 방어, 재생, 약화 | 디버프 누적, 재생/반사 연계 피해 |

## 주요 정수 목록

정수 시스템 규칙은 [[04_정수]]를 따른다.
밸런스 검토는 `rpg-tower-balance-sim.mjs`로 진행한다.
`rpg-tower-data.js`의 기존 정수 데이터는 구버전 구현 데이터이며, 새 100층 설계가 확정되면 교체 대상이다.

모든 몬스터는 서로 다른 정수를 가진다.
같은 계열의 정수라도 이름만 다른 복사본이 아니다.
각 정수는 공격 방식, 상태이상, 독, 흡혈, 버프, 반사, 쿨타임 환급, 방어막, 패시브 스탯 조합 중 서로 다른 메커니즘 조합을 가진다.

100층 전체 정수 목록과 등급별 효과는 [[../04_게임_데이터/01_정수_목록|정수 목록]]에서 관리한다.
