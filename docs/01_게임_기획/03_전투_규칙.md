# 전투 규칙

전투는 플레이어와 몬스터가 같은 규칙으로 행동하는 틱 기반 시뮬레이션이다.
이 문서는 전투 중 처리되는 규칙만 다룬다.

## 시간

- 1틱: 0.1초
- 10틱: 1초
- 전투는 틱 단위로 진행
- 매 틱마다 일반 공격 준비 상태와 정수 쿨타임 갱신

## 행동

- 한 캐릭터는 한 틱에 행동 1개만 실행 가능
- 행동 종류: 일반 공격, 정수
- 준비된 행동이 여러 개면 정수 슬롯 순서 우선
- 정수를 사용한 틱에는 일반 공격을 함께 실행하지 않음

## 일반 공격

- 일반 공격은 기본적으로 1초마다 준비
- 공격속도에 따라 일반 공격 준비 시간이 달라질 수 있음
- 일반 공격은 명중/회피 판정 대상
- 일반 공격은 치명타 발생 가능

## 정수

- 액티브 정수는 각자 정해진 쿨타임을 가짐
- 액티브 정수 사용은 일반 공격 준비 상태를 소모하지 않음
- 단, 정수도 행동 1개로 취급
- 패시브 정수는 장착 중 항상 적용

## 행동 우선순위

같은 틱에 양쪽 모두 행동 가능할 때의 처리 순서:

1. 플레이어 정수
2. 플레이어 일반 공격
3. 몬스터 정수
4. 몬스터 일반 공격

## 명중과 회피

- 일반 공격은 명중/회피 판정 대상
- 물리 피해와 마법 피해를 주는 정수도 기본적으로 명중/회피 판정 대상
- 고정 피해, 반사 피해, 지속 피해는 명중/회피 판정 대상이 아님

## 치명타

- 일반 공격은 치명타 발생 가능
- 물리 피해와 마법 피해를 주는 정수도 치명타 발생 가능
- 고정 피해, 반사 피해, 지속 피해는 치명타가 발생하지 않음
- 기본 치명타 피해: 150%
- 치명타 확률은 기본적으로 100%를 초과해도 추가 효과 없음
- 초과 치명타 확률 활용은 장비나 정수 효과로 별도 처리

## 피해와 방어

- 물리 피해는 물리 공격력을 기준으로 증가
- 마법 피해는 마법 공격력을 기준으로 증가
- 방어력 기반 피해는 방어력을 공격 계수처럼 사용
- 최종 방어는 받는 피해를 감소시킴
- 고정 피해는 방어의 영향을 받지 않음

기본 방어 계산:

```txt
피해 배율 = 100 / (100 + 최종 방어)
피해 감소율 = 최종 방어 / (100 + 최종 방어)
```

## 행동속도 보정

- 둔화와 가속은 공격속도 수치를 직접 바꾸지 않음
- 둔화와 가속은 행동속도 배율로 적용
- 행동속도 보정은 곱연산으로 중첩

```txt
최종 틱당 진행량 = 기본 틱당 진행량 × 행동속도 배율
```

## 전투 종료

- 한쪽의 체력이 0 이하가 되면 전투 종료
- 플레이어 체력이 0 이하가 되면 패배
- 몬스터 체력이 0 이하가 되면 승리

## 전투 시작 상태

- 자동 사냥은 이전 전투의 체력과 정수 쿨타임을 이어서 진행
- 탑 전투는 체력과 정수 쿨타임이 채워진 상태로 시작
